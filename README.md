# スマホケース用 情景プロンプト生成アプリ

スマホケースのイメージ画像を生成するときに使う、現実的な情景プロンプトをランダム生成するシンプルな Web アプリです。
生成される英語プロンプトは、Gemini に貼りやすいように指示のまとまりごとに整理しています。
元画像がある前提で、そのスマホケース自体の形状や色を変えない情景指示を出す用途を想定しています。

## できること

- スマホケース向けの商品画像プロンプトをランダム生成
- 1個から7個までの複数スマホケース構成に対応
- 横並び、ずらし配置、重なり配置などの並べ方を指定可能
- `自然`、`スポーツ`、`旅行`、`屋内`、`高級感`、`カフェ`、`日常`、`都会`、`仕事・作業`、`アウトドア`、`季節感` などのカテゴリで情景を絞り込み
- `写真の雰囲気` として `すっきり整った写真`、`日常っぽい写真`、`高級感のある写真`、`親しみやすい写真` を切り替え
- `主役の見せ方` として `商品単体寄り`、`情景寄り`、`バランス`、`カメラ開口部のクローズアップ` を切り替え
- 元画像のスマホケースの形状、色、開口部、デザインを維持する指示を含める
- 英語プロンプトと、その内容を確認しやすい日本語訳を並べて表示
- 日本語のシーン、光、構図、小物、雰囲気、追加条件を手で修正でき、その内容を英語プロンプトへ反映
- 生成したプロンプトのコピー

## このアプリの考え方

ありえない情景や不自然な組み合わせは避けて、商品画像として使いやすい範囲の要素だけをランダムに組み合わせます。
同時に、元画像に写っているスマホケース自体は変更しないという制約を常に含めます。

例えば以下のような方向です。

- 自宅デスク
- カフェテーブル
- ベッドサイド
- キッチンカウンター
- スタジオ撮影風
- 川辺のベンチや川沿いカフェ
- 砂漠の休憩スポットやロッジテラス
- テニスコート脇やクラブラウンジ
- ビーチサイド、山小屋、空港ラウンジ
- スパ、ヨガスタジオ、ガーデンベンチ

## ファイル構成

- `index.html`: 画面の構造
- `style.css`: 見た目とレイアウト
- `script.js`: ランダム生成ロジックとコピー処理
- `README.md`: 使い方の説明

## 使い方

1. `index.html` をブラウザで開きます。
2. ケース数、並べ方、カテゴリ、写真の雰囲気、主役の見せ方を選びます。
3. `再生成` を押してプロンプトを作ります。
4. 必要なら日本語編集欄の内容を修正します。
5. 修正内容は上の英語プロンプトに反映されます。
6. `コピー` を押して生成結果を使います。
